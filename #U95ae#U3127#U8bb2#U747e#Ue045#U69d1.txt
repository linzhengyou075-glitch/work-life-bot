Work Life V1.0.5.2 Final Stable（完整專案）

這是一份完整專案，可直接上傳至 GitHub／Render。

Build Command:
pip install -r requirements.txt

Start Command:
uvicorn main:app --host 0.0.0.0 --port $PORT

Render 必要環境變數：
LINE_LOGIN_CHANNEL_ID
LINE_LOGIN_CHANNEL_SECRET
LINE_CHANNEL_SECRET
LINE_CHANNEL_ACCESS_TOKEN
OWNER_LINE_USER_ID
SESSION_SECRET
DATABASE_URL（建議使用 PostgreSQL，避免重新部署後資料消失）

LINE Login Callback URL：
https://你的新Render網址/auth/line/callback

LINE Webhook URL：
https://你的新Render網址/webhook
